05/26/10 — Frogged.  It was too big to be practical.  (I never wore it.  What's worse than that!?) I don't think this yarn is meant to be used for a lacy beret... so I'm going to set it aside for another hat. Maybe a wintry one? 

04/04/10 — Finished!  As expected, the hat is a bit bigger than it would have been had I used the right yarn & needles... but c'est la vie.  It still fits!  It wears more like a rasta hat because it has such a slouchy interior... but I don't have dreads so I wear it like a beret ;)

04/03/10 — Despite my yarn and needles (which are not as recommended by the designer), I cast on 90 sts and am following the instructions exactly.
